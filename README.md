## FreeCvBuilder Header Layout

The CV header is composed of three columns, from left to right:

1. **Header column 1 – Picture column**  
   Contains the profile picture.
2. **Header column 2 – Name and title column**  
   Contains the full name and professional title.
3. **Header column 3 – Contact column**  
   Contains all contact details (email, phone, location, website, LinkedIn, etc.).

Whenever this documentation or future instructions refer to **"header column 2"**, it means the **name and title column** (full name and title).
